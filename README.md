# AI CV Sorter — Final Project Jobdesk Site

Website informasi pembagian jobdesk untuk Final Project AI for Business - Topik 20.

## 🚀 Cara Deploy ke Vercel

1. Push repo ini ke GitHub
2. Buka [vercel.com](https://vercel.com) → Login → Add New Project
3. Import repo dari GitHub
4. Klik Deploy — selesai!
