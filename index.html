Topik 20 — Sistem Perekrutan Cerdas Terintegrasi

💡
Baca dulu sebelum mulai! Pilih tab sesuai nama kamu di navigasi atas untuk melihat instruksi jobdesk-mu secara lengkap.
🎯 Latar Belakang Masalah

HRD perusahaan sering menerima ratusan CV per posisi yang masuk via email. Proses membaca dan menyortir secara manual sangat memakan waktu, tidak konsisten, dan kandidat yang lolos tidak langsung mendapat jadwal interview.

✅ Tujuan Sistem

Membangun sistem otomasi yang mampu:

  • Membaca CV PDF yang dikirim via email secara otomatis
  • Menilai & memberi skor CV menggunakan AI (dibandingkan Job Description di Notion)
  • Mengirim email penerimaan + link booking interview ke kandidat yang lolos
  • Mengirim email penolakan personal (bukan template generik) ke yang tidak lolos
  • Mencatat semua data pelamar ke Airtable secara otomatis

Jobdesk 2 — Setup Akun & Credential

PIC: Fawwaz & Bagus · Deadline: Pekan M3 Mei

⚠️
Jobdesk ini adalah fondasi dari seluruh project. Tim workflow (Afif & Shinta) tidak bisa mulai sebelum semua credential selesai dan dikirim ke grup!

📋 5 Service yang Harus Disiapkan

A. 📧 Gmail (Akun HR Khusus)

Buat akun Gmail baru yang akan menjadi inbox HR — semua CV pelamar dikirim ke sini.

1. Buat akun Gmail baru
Email disarankan: hr.aicvsorter2026@gmail.com
2. Hubungkan ke n8n
Di n8n → Settings → Credentials → Add → Gmail OAuth2 API → Connect → Login akun HR → Save dengan nama "Gmail HR - CV Sorter"
B. 🤖 Groq API (AI Model — GRATIS)
1. Daftar Groq
Buka console.groq.com → Login dengan Google
2. Buat API Key
Dashboard Groq → API Keys → Create API Key → Nama: "AI CV Sorter Final Project" → ⚠️ SEGERA COPY! Hanya tampil sekali!
3. Tambahkan ke n8n
Settings → Credentials → Add → cari "Groq" → paste API key → Save
C. 📝 Notion (Job Description Database)
1. Buat halaman Job Description
Di Notion, buat halaman baru → Judul: "Job Description - Data Analyst"
2. Isi konten halaman persis seperti ini:
Posisi: Data Analyst Perusahaan: PT. Teknologi Nusantara == KUALIFIKASI WAJIB == - S1 Informatika / Statistik / Matematika / Teknik - Pengalaman minimal 1 tahun di bidang data analysis - Menguasai SQL (query, join, aggregation) - Menguasai Python atau R - Familiar dengan Tableau / Power BI / Google Data Studio == KUALIFIKASI TAMBAHAN == - Cloud (AWS, GCP, Azure) - Machine Learning dasar - Git / version control == KRITERIA SKOR == - Technical Skills : 40 poin (Python/R, SQL, Visualisasi) - Pengalaman Kerja : 30 poin - Pendidikan : 15 poin - Soft Skills : 15 poin Score >= 65 = LOLOS | Score < 65 = GAGAL
3. Buat Integration & copy API key
notion.so/my-integrations → New Integration → Nama "n8n AI CV Sorter" → Submit → Copy Internal Integration Secret
4. Connect Integration ke halaman
Kembali ke halaman Job Description → klik ••• → Add connections → pilih integration yang baru dibuat
5. Copy Page ID
Dari URL: notion.so/workspace/[INI PAGE ID-NYA]?v=... → copy bagian yang dibold
D. 🗄️ Airtable (Database Rekrutmen)
1. Buat Base & Table di Airtable
airtable.com → Add a base → Start from scratch → Nama: "Rekrutmen AI CV Sorter" → Rename tabel: "Data Pelamar"
2. Buat kolom-kolom ini:
Nama FieldTipe FieldOpsi
Nama PelamarSingle line text—
Email PelamarEmail—
Posisi DilamarSingle line text—
Score AINumber—
StatusSingle selectLOLOS, GAGAL, NO_PDF
KelebihanLong text—
KekuranganLong text—
Ringkasan AILong text—
Tanggal MasukDate—
Status EmailSingle selectSent, Pending, Failed
Link CalendarURL—
3. Buat API Token
airtable.com/create/tokens → Create token → Nama "n8n CV Sorter" → Scopes: data.records:read + data.records:write → Access: pilih base tadi → Create → ⚠️ SEGERA COPY!
4. Copy Base ID
Dari URL: airtable.com/appXXXXXXXXXXXXXX/... → copy yang dimulai app...
E. 📅 Google Calendar / Calendly (Link Interview)

Buat link booking jadwal interview yang akan dikirim ke kandidat yang lolos.

Opsi A — Google Calendar:

  • calendar.google.com → klik + → New appointment schedule
  • Judul: "Interview Data Analyst - PT. Teknologi Nusantara"
  • Durasi: 45 menit
  • Atur ketersediaan waktu
  • Copy link booking yang dihasilkan

Opsi B — Calendly (Lebih mudah):

  • calendly.com → Daftar gratis
  • Create event → "Interview Data Analyst"
  • Durasi 45 menit, atur ketersediaan
  • Copy link dari dashboard

Jobdesk 3 — Pembuatan Workflow n8n

PIC: Afif & Shinta · Deadline: Pekan M4-M5 Mei

🗺️ 9 Node yang Harus Dibangun

Node 1: Gmail Trigger
Pantau inbox HR, picu workflow saat CV masuk
↓
Node 2: IF — Cek PDF
Ada attachment PDF? Ya → lanjut | Tidak → catat NO_PDF
↓ Ya
Node 3: Extract from File
Baca teks dari PDF CV
↓
Node 4: HTTP Request → Notion
Ambil Job Description dari Notion
↓
Node 5: AI Agent (Groq)
Scoring CV vs Job Desc → output JSON
↓
Node 6: Switch LOLOS / GAGAL
Routing berdasarkan status dari AI
Node 7A: Generate Accept Email
Node 8A: Gmail Send (Accept + Calendar)
Node 9A: Airtable (Simpan LOLOS)
Node 7B: Generate Rejection Email
Node 8B: Gmail Send (Rejection)
Node 9B: Airtable (Simpan GAGAL)

📝 Konfigurasi Detail Per Node

🔵 Node 5 — System Prompt AI (Groq)

Copy-paste prompt ini ke field "System Message" pada AI Agent node:

Kamu adalah AI Recruitment Specialist yang bertugas menilai CV kandidat secara objektif. JOB DESCRIPTION: Posisi: Data Analyst | PT. Teknologi Nusantara Wajib: S1 Informatika/Statistik/Matematika, min. 1 thn pengalaman data, SQL, Python/R, tools visualisasi Plus: Cloud, Machine Learning, Git Skor: Technical 40 poin | Pengalaman 30 poin | Pendidikan 15 poin | Soft Skills 15 poin CV KANDIDAT: {{ $('📄 Extract Teks dari CV PDF').item.json.text }} Tugasmu: 1. Analisis kesesuaian CV dengan kriteria di atas 2. Berikan skor total 0-100 3. Status: "LOLOS" jika score >= 65 | "GAGAL" jika score < 65 4. Identifikasi 3 kelebihan dan 2 kekurangan 5. Ekstrak nama lengkap dan email dari CV KEMBALIKAN HANYA JSON VALID (tanpa markdown): {"nama_pelamar":"string","email_pelamar":"string","posisi_dilamar":"Data Analyst","score":number,"status":"LOLOS atau GAGAL","kelebihan":"string","kekurangan":"string","ringkasan_ai":"string"}
✉️ Prompt — Generate Accept Email (Node 7A)
Buat email penerimaan seleksi awal yang hangat dan profesional dalam Bahasa Indonesia. Nama kandidat : {{ $('🤖 AI Scoring').item.json.output.nama_pelamar }} Posisi : Data Analyst Kelebihan : {{ $('🤖 AI Scoring').item.json.output.kelebihan }} Link interview: [MASUKKAN LINK CALENDAR DARI JOBDESK 2] Syarat: Sapa nama, sampaikan lolos, sebut kelebihan spesifik, minta booking via link, maks 150 kata. Kembalikan HANYA isi email (tanpa subject).
✉️ Prompt — Generate Rejection Email (Node 7B)
Buat email penolakan yang sopan, personal, dan memotivasi dalam Bahasa Indonesia. Nama kandidat : {{ $('🤖 AI Scoring').item.json.output.nama_pelamar }} Posisi : Data Analyst Kekurangan : {{ $('🤖 AI Scoring').item.json.output.kekurangan }} Syarat: Sapa nama, tolak dengan halus, beri saran konstruktif dari kekurangan, tutup dengan motivasi, maks 150 kata. Kembalikan HANYA isi email (tanpa subject).
